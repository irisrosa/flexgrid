# Merengue

Merengue is a small library of React components to facilitate creating page grids.

## Usage

```
<MerengueBox>
  <Block>...</Block>
  <Block>...</Block>
  <Block>...</Block>
</MerengueBox>
```

or

```
<MerengueBox blocks={[...]} />
```
